### Tiny OpenGL Engine

#### The project and wiki is work in progress.
##### Currently features:
- Model loading
- Blinn-Phong Lighting
- Multiple lights
- HDR
- Bloom
- Depth of Field
##### Next features:
- Shadows
- Normal mapping / Parallax Mapping
