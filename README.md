### Tiny OpenGL Engine

#### The project and wiki is work in progress.
##### Currently features:
- Model loading
- Blinn-Phong Lighting
- Multiple lights
- HDR
- Bloom
- Depth of Field
- ImGui
- Shadowmapping
- Cascaded Shadowmapping

##### Next features:
- Normal mapping / Parallax Mapping
- PBR
